When to use: running a review cycle, coaching a manager to write a fair review, or designing calibration so ratings are consistent and defensible. Keep feedback specific, evidence-based, and tied to expectations for the person's level.

METHOD
1. Set the frame: state the review period, the competencies/goals being assessed, and the rating scale with clear definitions for each level.
2. Self-assessment: give the employee prompts covering accomplishments (with impact), goal progress, strengths, growth areas, and support needed. Ask for evidence, not adjectives.
3. Manager template: for each competency/goal, capture concrete examples, a rating with justification, and forward-looking development actions. Separate performance-in-role from potential.
4. Gather multi-source input: incorporate peer/stakeholder feedback where available; attribute themes, not gossip, and check for recency or halo bias.
5. Draft the rating: map evidence to the scale for the person's level; write the summary so it would hold up if read aloud to the employee.
6. Calibrate: compare ratings across a peer group, surface inconsistencies and bias patterns, adjust with documented rationale, and finalize the distribution.

OUTPUT FORMAT
- Review frame (period, competencies, rating scale definitions)
- Self-assessment template (prompts)
- Manager review template (per-item: examples, rating, rationale, development)
- Calibration guide (comparison method, bias checks, adjustment log)
- Final summary structure + delivery tips
