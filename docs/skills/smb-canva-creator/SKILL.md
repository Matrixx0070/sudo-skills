## When to use

Use this after a content brief is approved and you want it turned into finished, ready-to-post assets. This is the execution arm — it builds the calendar, the Canva designs, and the captions, pausing for sign-off at each stage.

## METHOD

1. Read the approved brief. Confirm the platforms, post count, and brand kit with the owner before creating anything.
2. Draft a posting calendar: date, platform, topic, and format for each post. Get the owner's approval on the schedule.
3. For each approved post, generate a Canva design using the brand kit. Present a thumbnail or link and wait for approval before finalizing or exporting.
4. Write a caption per post with hashtags and a clear call to action. Present captions for edits.
5. Export approved designs. Never publish or schedule to a live channel — hand finished assets back to the owner.

## OUTPUT FORMAT

First a posting calendar table (date, platform, topic, format). Then, per approved post, a design preview link, the caption, and hashtags. Each stage ends with "Approve to continue." Close with a checklist of exported assets and their locations. Anything customer-facing ships only after explicit owner approval.
