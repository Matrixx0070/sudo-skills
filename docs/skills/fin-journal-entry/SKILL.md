# Journal Entry

## When to use
Use this to draft a journal entry that will pass review the first time — accruals, prepaid amortization, depreciation, revenue recognition, or reclasses — with the debits, credits, and supporting rationale an approver needs.

## METHOD
1. **Identify the transaction and type.** State what happened, the entry type (accrual, deferral, depreciation, rev-rec, reclass), and the period it belongs to.
2. **Apply the accounting treatment.** Cite the driver: matching principle for accruals, straight-line or the chosen method for depreciation, ASC 606 five-step for revenue.
3. **Compute the amount.** Show the calculation (e.g., cost less salvage over useful life; contract value times % complete or ratable portion).
4. **Build the entry.** List each account with its number, debit or credit, and amount; every entry must have debits equal credits.
5. **Attach support.** Reference the invoice, contract, schedule, or accrual worksheet backing each line.
6. **Plan the reversal.** For accruals and deferrals, state the reversal date and mechanism.
7. **Self-check.** Confirm the entry balances, hits the correct period, and uses valid accounts.

## OUTPUT FORMAT
- **Entry description / type / period / posting date.**
- **Accounting rationale:** principle or standard applied.
- **Calculation:** shown step by step.
- **Journal entry table:** account, account #, debit, credit — with column totals proving debits = credits.
- **Supporting documents referenced.**
- **Reversal instruction (if applicable).**
- **Reviewer notes.**
