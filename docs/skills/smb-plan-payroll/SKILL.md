# Plan Payroll

## When to use
Use this in the days before a payroll run when the owner needs certainty that cash will cover it — and, if it's tight, a prioritized list of collections that would close the gap.

## METHOD
1. **Set payroll target.** State the payroll date and total amount due including employer taxes and any contractor payments.
2. **Project cash to that date.** Start from today's balance, add expected inflows (AR by realistic pay date), subtract non-payroll obligations due before then.
3. **Compute coverage.** Compare projected available cash on payroll date against the payroll amount; state surplus or gap and the buffer left over.
4. **Rank collectible invoices.** If tight, sort overdue invoices by likelihood-to-collect-fast × amount to show which follow-ups most improve coverage.
5. **List levers.** Note other options — delay discretionary AP, draw on a line of credit, owner contribution — with amounts.
6. **Give a go / at-risk call.** State plainly whether payroll runs comfortably.

Do not chase invoices, delay payments, or draw credit yourself; present the plan for owner approval — payroll and money decisions are the owner's.

## OUTPUT FORMAT
- **Payroll date / amount due.**
- **Cash projection to date:** starting, inflows, outflows, available.
- **Coverage verdict:** surplus/gap, buffer, go / at-risk.
- **Ranked collection targets:** invoice, amount, collect-fast likelihood.
- **Other levers:** option, amount.
- **Recommendation (for owner approval).**
