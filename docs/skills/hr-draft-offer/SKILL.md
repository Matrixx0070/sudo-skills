When to use: a candidate has verbally accepted or a hiring manager has approved an offer and you need a written letter ready to send. Produce a professional, accurate document — never invent numbers, dates, or legal terms; flag anything unconfirmed.

METHOD
1. Collect inputs: candidate name, role title, level, manager, team, start date, employment type (full-time/part-time/contract), work location/remote status, and reporting line.
2. Assemble compensation: base salary (and pay period), sign-on bonus (with clawback window if any), target annual bonus, equity grant (type, amount, vesting, cliff), and benefits summary.
3. State conditions: at-will language (jurisdiction-appropriate), background check, work authorization/I-9 or local equivalent, and any offer expiration date.
4. Add key terms: relocation, confidentiality/IP assignment, PTO policy, and probation period if applicable.
5. Set tone: warm and welcoming opener, precise on all numbers, clean acceptance instructions (signature + date + deadline).
6. Verify before delivery: cross-check every figure and date against the approved requisition; mark [CONFIRM] on any placeholder and route legal-sensitive clauses to counsel.

OUTPUT FORMAT
- Offer letter (ready to send), sections: greeting, position, start date, compensation, equity, benefits, contingencies, at-will/terms, acceptance block
- Placeholder list: every [CONFIRM] item needing sign-off
- Reviewer notes: clauses to route to legal/finance before sending
